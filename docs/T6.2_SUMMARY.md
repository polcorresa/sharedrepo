# T6.2 Summary: Tree View

## Overview
Implemented the file tree visualization and management UI.

## Components

### 1. Hook (`apps/web/src/hooks/useTree.ts`)
- **`useTree`**:
  - Fetches tree data using React Query (`GET /api/repos/:slug/tree`).
  - Provides mutations for:
    - `createFolder` / `createFile`
    - `renameFolder` / `renameFile`
    - `deleteFolder` / `deleteFile`
  - Invalidates queries on success to refresh the view.

### 2. Component (`apps/web/src/components/TreeView.tsx`)
- **Recursive Rendering**: Displays nested folder structure.
- **Interactions**:
  - Expand/Collapse folders.
  - Select files (calls `onSelectFile`).
  - Context actions (buttons for now) to create/delete nodes.
- **State**: Manages expanded folders locally.

### 3. Integration (`apps/web/src/components/RepoEditor.tsx`)
- Integrates `TreeView` into the sidebar.
- Manages `activeFile` state to show the selected file in the main area.
